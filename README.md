## 🏛️ Project Overview

This repository hosts the source code for [john-croft.com](https://john-croft.com), a bespoke digital space designed to showcase professional work in software engineering while serving as a repository for philosophical thoughts and explorations.

The site is built with a focus on **Visual Excellence** and **Technical Precision**, bridging the gap between a traditional portfolio and a modern digital garden.

## ✨ Design Philosophy

Inspired by high-end design houses like Cuberto and Ferrumpipe, this site implements a "Premium Aesthetic" characterized by:

- **Massive Typography**: Bold, expressive headings that command attention.
- **Tactile Interaction**: Smooth, GSAP-driven scroll-reveal animations and interactive elements.
- **Luxurious Whitespace**: Generous breathing room to focus on content.
- **Extreme Geometry**: Large 80px border radiuses for a soft, modern, and high-end feel.
- **Dynamic Theming**: A sophisticated theme toggle for seamless light and dark mode transitions.

## 🚀 Core Features

- **Substack Integration**: Real-time fetching and displaying of recent posts from [Lately Thinking](https://latelythinking.substack.com).
- **Interactive Portfolio**: A curated showcase of engineering projects and philosophical work.
- **Responsive Animations**: Performant scroll-triggered animations using GSAP and ScrollTrigger.
- **Digital Garden**: A space for evolving thoughts, reading lists, and personal insights.
- **Optimized Performance**: Fast load times through Jekyll's static site generation.

## 🛠️ Tech Stack

- **Core**: [Jekyll](https://jekyllrb.com/) (Ruby)
- **Styling**: Vanilla SCSS with a custom-built design system.
- **Animations**: [GSAP](https://greensock.com/gsap/) (GreenSock Animation Platform) for high-performance motion.
- **Scripts**: Modular Vanilla JavaScript.
- **Components**: Typed.js for dynamic headers, custom carousel implementations.

## 🛠️ Local Development

### Prerequisites
- **Ruby** (v3.0.0 or higher recommended)
- **Bundler** (`gem install bundler`)

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/jmcroft7/philosophy-website.git
   ```
2. Install dependencies:
   ```bash
   bundle install
   ```
3. Start the local server:
   ```bash
   bundle exec jekyll serve
   ```
4. Access the site at `http://localhost:4000`

## 🌐 Deployment

The site is automatically deployed to **GitHub Pages** whenever changes are pushed to the `main` branch.
